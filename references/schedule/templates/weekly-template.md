# 周报输出模板

> **Token 预算**：~500 tokens
> **输出形式**：推送（不保存文件）

---

## 结构定义

| Section | Token | 内容要求 | 格式规范 |
|---------|-------|---------|---------|
| **标题** | ~10 | 【Life Index 周报】{周次} | 固定格式 |
| **本周概要** | 50-80 | 3-4 句话总结本周核心内容 | 纯文本 |
| **要点速览** | 100-150 | 5-7 个关键事项 | `• ` 开头 |
| **AI 建议** | 50-80 | 2-4 条建议 | `→ ` 开头 |
| **要事提醒** | 0-50 | 待跟进事项 | `🔔 ` 开头 |
| **本周热点** | 100-150 | Top 5 国际热点，每条 1 句话 | 编号列表 |
| **目标进度** | 30-50 | 年度目标进度更新 | 表格或文字 |

## 热点获取规范

**执行方式**：
1. 使用 Agent 网络搜索能力
2. 搜索关键词：`this week world news`、`本周国际热点`
3. 筛选 Top 5，每条 1 句话描述

**超时**：30 秒

**失败处理**：
```
IF 热点获取失败:
  → 省略热点部分
  → 在报告中备注："热点获取失败，已跳过"
```

## 验证清单

```
⛔ STOP. 推送前验证：

- [ ] 本周概要：3-4 句话（50-80 tokens）✓
- [ ] 要点速览：5-7 条（100-150 tokens）✓
- [ ] AI 建议：2-4 条（50-80 tokens）✓
- [ ] 要事提醒：如有待办则列出 ✓
- [ ] 本周热点：Top 5（如获取成功）✓
- [ ] 目标进度：如有更新则列出 ✓
- [ ] 无敏感信息 ✓
```
