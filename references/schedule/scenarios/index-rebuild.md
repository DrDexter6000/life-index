# 每月索引维护场景指南

> **场景**：全量检查索引完整性，重建（如需），向量全量更新
> **触发**：每月1日 03:30

---

## 背景说明

**与每日维护的区别**：

| 维度 | 每日维护 | 每月维护 |
|------|---------|---------|
| **扫描范围** | 昨日日志 | 当月所有日志 |
| **检查深度** | 一致性检查 | 完整性检查 |
| **修复方式** | 增量修复 | 全量重建（如需） |
| **向量更新** | 增量 | 全量 |

---

## 触发条件

| 类型 | 条件 | 行为 |
|------|------|------|
| **Immediate** | 系统时间到达每月1日 03:30 | 自动执行 |

---

## 执行流程

### Step 1: 扫描当月日志

**命令**：
```bash
python tools/search_journals.py --month {LAST_MONTH} --limit 5000
```

**参数说明**：
- `{LAST_MONTH}`: 上月年月，格式 `YYYY-MM`（每月1日执行，检查刚结束的月份）
- 示例: `python tools/search_journals.py --month 2026-02 --limit 5000`

**输出解析**：
```json
{
  "success": true,
  "data": [
    {"title": "...", "path": "Journals/2026/02/life-index_2026-02-15_001.md", ...}
  ]
}
```

**记录结果**：
```
上月日志数量：{COUNT}
日志路径列表：[...]
```

---

### Step 2: 检查索引完整性

**检查项**：

| 检查项 | 说明 | 通过条件 |
|--------|------|----------|
| **主题索引完整性** | 所有主题都有对应的索引文件 | 无遗漏 |
| **项目索引完整性** | 所有项目都有对应的索引文件 | 无遗漏 |
| **标签索引完整性** | 所有标签都有对应的索引文件 | 无遗漏 |
| **索引引用完整性** | 每篇日志都被正确引用 | 无孤立日志 |
| **无孤立索引** | 索引文件引用的日志都存在 | 无无效引用 |
| **月度摘要存在** | 当月有 `monthly_abstract.md` | 文件存在 |

**执行方式**：
1. 遍历当月所有日志
2. 提取所有 topic/project/tags
3. 检查对应的索引文件
4. 检查每篇日志是否被正确引用

**结果判断**：
```
IF 所有检查项通过:
  → 记录 "索引完整性检查通过"
  → Proceed to Step 4（向量全量更新）

IF 发现问题:
  → 记录问题详情
  → Proceed to Step 3（全量重建）
```

---

### Step 3: 全量重建（如需）

**触发条件**：Step 2 发现完整性问题

**命令**：
```bash
python tools/build_index.py --rebuild
```

**参数说明**：
- `--rebuild`: 全量重建模式
- 重建所有 by-topic 索引和向量索引

**执行时间**：可能较长（取决于日志数量）

**重建后验证**：
```
IF 重建成功:
  → 记录重建详情
  → Proceed to Step 4

IF 重建失败:
  → 记录错误
  → 通知用户
  → 结束执行
```

---

### Step 4: 向量全量更新

**前提条件**：已安装 `sentence-transformers`

**命令**：
```bash
python tools/build_index.py --rebuild
```

**说明**：
- 全量重建向量索引
- 处理所有日志的嵌入向量

**跳过条件**：
```
IF sentence-transformers 未安装:
  → 记录 "向量索引跳过：未安装 sentence-transformers"
  → 继续执行 Step 5
```

---

### Step 5: 记录结果

**成功（无问题）**：
```
日志记录："[{NOW}] Monthly index maintenance completed. Month: {MONTH}, Journals: {COUNT}, Issues: 0"
```

**成功（发现并修复问题）**：
```
日志记录："[{NOW}] Monthly index maintenance completed. Issues found: {N}, Rebuilt: true"
```

**失败**：
```
日志记录："[{NOW}] Monthly index maintenance failed: {ERROR}"
通知用户
```

## 验证规则

### 完整性检查清单

```
验证项：
- [ ] 上月日志已全量扫描 ✓
- [ ] 所有主题索引文件存在 ✓
- [ ] 所有项目索引文件存在 ✓
- [ ] 所有标签索引文件存在 ✓
- [ ] 无孤立日志（每篇都被引用）✓
- [ ] 无孤立索引（引用都有效）✓
- [ ] 月度摘要存在 ✓
- [ ] 向量索引已全量更新（如已安装依赖）✓
```

---

## 错误处理

| 场景 | 处理 | 重试 |
|------|------|------|
| 工具执行失败 | 记录错误，通知用户 | 下月执行 |
| 发现完整性问题 | 全量重建 | 是 |
| 重建失败 | 记录错误，通知用户 | 否 |
| 向量更新跳过 | 记录原因，继续执行 | N/A |
